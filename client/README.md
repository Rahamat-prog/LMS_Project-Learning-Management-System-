# LMS Frontend

This folder contains the frontend application for the LMS project.

## What is done so far

- Initialized the frontend app with **Vite** and **React**.
- Installed core frontend dependencies:
  - `react`
  - `react-dom`
  - `tailwindcss`
  - `@tailwindcss/vite`
- Set up development and build scripts in `package.json`.
- Added linting support with **ESLint** and React-related plugins.
- Created the basic `src/` structure with:
  - `main.jsx`
  - `App.jsx`
  - `index.css`
  - `App.css`
  - `assets/`

## How to run

1. Open a terminal in the `client` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Available scripts

- `npm run dev` - Start the Vite development server.
- `npm run build` - Build the frontend for production.
- `npm run preview` - Preview the production build locally.
- `npm run lint` - Run ESLint across the frontend source.

## Notes

- The frontend scaffold is ready and the project is configured for building the LMS UI.
- Backend integration and specific page/component implementation are not yet completed in this folder.
